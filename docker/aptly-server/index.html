deb.openswitch.net

This is the repository for the OpenSwitch OPX Operating System. You can install an official release or a development release.

To install an official release (replace 2.0 with desired release), use this repository

deb http://deb.openswitch.net/ 2.0 main

Otherwise you can install packages from the stable, testing, or unstable repositories.

deb http://deb.openswitch.net/ stable main
deb-src http://deb.openswitch.net/ stable main

deb http://deb.openswitch.net/ testing main
deb-src http://deb.openswitch.net/ testing main

deb http://deb.openswitch.net/ unstable main
deb-src http://deb.openswitch.net/ unstable main

After that you can import our key and install OPX as any other software package:

gpg --keyserver pgp.mit.edu --recv-keys AD5073F1
gpg -a --export AD5073F1 | apt-key add -

apt-get update
apt-get install opx-dell-vm
